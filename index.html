Présentation du projet

L'idée

L'idée originale du projet était de réaliser une carte de restaurant en répartissant les différentes parties dans plusieurs pages. Après un certains temps de réflexion, la répartition Boissons / Plats / Desserts fût le choix le plus optimal.

Le format

Chaque membre du groupe s'est donc mis à écrire sa propre page en suivant tout de fois une même logique. Les différentes catégories sont rangées dans un tableau, les produits sont stockés dans une liste contenant à la fois le titre, une description et un prix.

Serveur web

Outil utilisé

Nous avons choisis d'utiliser NGINX comme serveur web. En effet, c'est est un logiciel moderne, simple d’utilisation avec un système de configuration semblable à un langage de programmation. Il permet une installation rapide sur plusieurs supports et de nombreuses fonctionnalités simples et avancés.

Problèmes rencontrés

Faisant déjà l'usage régulièrement de l'outils, nous n'avons pas eu beaucoup de problèmes. Toutefois, ce n'était pas le cas pour tous. En effet, après une arrivée en cours d'année de Paul, Nous avons dû lui expliquer comme fonctionne l'outils. Ce n'était pas chose facile.
